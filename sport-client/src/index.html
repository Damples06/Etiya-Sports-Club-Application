About the project

This application was developed during an internship at Etiya. It serves as a comprehensive tool for managing sports club operations, including user roles, course bundles, member arrivals, and calendar events. Leveraging Java, Spring Framework, and Angular, it provides a secure and efficient way to handle various club activities.

Why this project done

The Sports Club Application was created to address the need for an integrated solution to manage sports clubs more effectively. The goal was to combine various management tasks into a single platform, improving efficiency and ease of use. By utilizing modern technologies, the project aims to enhance the operational capabilities of sports clubs and streamline their day-to-day processes.
